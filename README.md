# chat
### netty5 实现聊天室

```java
1.chatServer：服务器端 netty5+protobuf+spring4,包含推送功能。

推送页地址：http://localhost:8080/chatServer/chat/pushIndex

2.javaClient：java swing 图形界面客户端。

3.AndroidClient：android客户端（delay）。

4.使用protobuf传输数据。
```
### compatibility：
```java
The java version >=1.6.
```
